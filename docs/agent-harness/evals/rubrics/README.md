# Rubrics

Phase 2A에서는 공통 rubric 디렉터리만 만든다. 실제 rubric 문서는 Phase 2B 이후 structured eval과 grader가 필요해질 때 추가한다.

현재 필요한 공통 rubric 후보:

- analysis finding quality
- routing decision quality
- behavior-preservation evidence quality
- bug-fix reproduction quality
- safety report quality

Rubric 문서는 `docs/agent-harness/evals/schemas/skill-eval.schema.md`의 `rubric` expectation을 채점할 때만 사용한다.
